# titanic-survival_nalysis
##Project Overview

This project develops a Logistic Regression model to predict passenger survival on the Titanic, based on various passenger features.

##Technologies Used

#Python
Pandas: Data manipulation
Matplotlib, Seaborn: Data visualization
Scikit-learn: Machine learning (model selection, training, evaluation)
Model Performance

##Training Accuracy: 80.76%
Test Accuracy: 78.21%
This indicates a reasonable prediction capability with slight overfitting.
